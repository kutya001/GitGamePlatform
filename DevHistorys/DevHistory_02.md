# История разработки - Версия 1.0.1 (Hotfix Deployment)

## Исправление деплоя на GitHub Pages
**Проблема:**
- При открытии сайта `https://kutya001.github.io/GitGamePlatform/` отображался пустой экран.
- В консоли отсутствовали ошибки выполнения JS, так как скрипт не загружался вовсе.
- **Причина 1:** В `index.html` отсутствовал тег `<script type="module" src="./index.tsx"></script>`, необходимый для запуска Vite-приложения.
- **Причина 2:** Отсутствовала конфигурация `base` в Vite, из-за чего сборщик предполагал, что сайт лежит в корне домена, а не в папке `/GitGamePlatform/`.

**Решение:**
1. **index.html**:
   - Добавлен тег `<script>` указывающий на точку входа `./index.tsx`.
   - Удален `react-router-dom` из `importmap`, так как используется кастомный роутер в `store.ts`.
2. **vite.config.ts**:
   - Создан файл конфигурации.
   - Установлен параметр `base: '/GitGamePlatform/'`, что позволяет корректно генерировать пути к ассетам (JS/CSS) внутри поддиректории на GitHub Pages.
